HDTickets FTP Deployment Instructions
=====================================

Files to Upload:
- hdtickets-production-complete.zip (40MB) - Main application package

FTP Connection Details Needed:
- Host: hdtickets.polascin.net (or your hosting server IP)
- Username: [Your hosting username]
- Password: [Your hosting password]
- Protocol: SFTP (recommended) or FTP
- Port: 22 (SFTP) or 21 (FTP)

Upload Steps:
1. Connect to your hosting server via FTP/SFTP
2. Navigate to your home directory (usually /home/yourusername/)
3. Upload hdtickets-production-complete.zip to the root of your account
4. Log into your hosting control panel
5. Use File Manager to extract the ZIP file
6. Follow the remaining steps in DEPLOYMENT-SHARED-HOSTING.md

Important Notes:
- The ZIP file contains all application files including built assets
- Make sure to extract it in your home directory, not public_html directly
- After extraction, you'll need to configure the web directory structure
- Don't forget to set up your database before testing

Next Steps After Upload:
1. Extract the ZIP file
2. Configure environment variables (.env)
3. Set up database
4. Configure web directory (public_html)
5. Install PHP dependencies (if composer is available)
6. Test the application
